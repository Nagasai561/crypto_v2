Instructions
To encrypt as text
  1. Tick the "Encrypt as text" checbox
  2. Choose the method you want to use
  3. If you choose "User generated", please make sure that key is a positive integer less than 60000
  4. Enter your message in the appropriate input field
  5. If you are using "User generated key", then enter a key in the appropriate field
  6. Press "Encrypt" button
  7. You will see cipher text in it's appropriate box
  8. To decypt a message, you need to know ciphered text, it's corresponding key (Note that in "No key" method, key isn't required) and the method by which it was encrypted
  9. Again tick "Encrypt as text" and choose the correct method by which ciphertext was obtained
  10. Enter the cipher text in appropriate textbox
  11. Enter the key in the appropriate textbox (No need to do this step if you have chosen "No key" method)
  12. Press "Decrypt" button
  13. You will see the decrypted text in the "Plaintext box"
To encrypt as image
  1. Tick the "Encrypt as image" checkbox
  2. Choose the method you want to use
  3. Type your message in the indicated box
  4. Press "Encrypt" button
  5. Download the noisy image you see on side
  6. Send that image to anyone you like
  7. To decrypt text from image, you need to have the image on your device
  8. Click on "Choose File" and upload that image
  9. Press "Decrypt" button
  10. Decrypted text will be displayed on the corressponding textbox
